# Roadmap de Carreira em Software Engineering

Plano interativo de 12 semanas com navegação fluida, checklist de tarefas e temas claro/escuro. 

**Desenvolvido por:** Lucasdeveng

## 🚀 Versões Disponíveis

### 1. **Versão HTML Pura** (Atual - Sem dependências)
A forma mais simples de usar: abra `index.html` no navegador.

- ✅ Sem setup necessário
- ✅ Funciona offline
- ✅ Temas claro/escuro
- ✅ Persistência local automática
- ⚙️ Vanilla JavaScript

**Como usar:**
```bash
Abra index.html no navegador
```

### 2. **Versão React** (Com estado gerenciado)
Componente React otimizado com `useState` e `useEffect`.

- ✅ Gerenciamento de estado com React Hooks
- ✅ Variáveis CSS dinâmicas (`--color-bg-primary`, `--color-card`, etc)
- ✅ Build otimizado com Vite
- ⚙️ Requer Node.js

**Como usar:**
```bash
npm install
npm run dev  # Desenvolvem
npm run build # Produção
```

Para mais detalhes sobre a versão React, veja [REACT_VERSION.md](REACT_VERSION.md).

## 📋 Funcionalidades

- 12 semanas de conteúdo estruturado
- Checklist interativo com progresso
- Navegação entre semanas com animações fluidas
- Temas claro (branco) e escuro
- Persistência automática no localStorage
- Botões com efeito hover e transições suaves
- Design responsivo

## 🎨 Temas

### Claro (Padrão)
- Fundo: Branco
- Texto: Cinza escuro
- Cards: Branco com bordas sutis

### Escuro
- Fundo: Cinza-azulado muito escuro
- Texto: Branco translúcido
- Cards: Cinza-azulado

Alterne entre temas clicando no botão "Tema escuro" / "Tema claro".

## 💾 Persistência Local

O app salva automaticamente:
- Última semana aberta
- Tarefas marcadas como concluídas
- Tema selecionado

Seus dados são preservados ao recarregar a página.
